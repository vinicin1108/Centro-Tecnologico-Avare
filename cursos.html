Tecnologia

Análise e Desenvolvimento de Sistemas

3 anos · 6 períodos · 1.800 horas

Formação completa em desenvolvimento full-stack, computação em nuvem, ciência de dados e DevOps.

1 SEM

1º Período — Fundamentos

300h

Base sólida em lógica, algoritmos e pensamento computacional.

  • Teoria Introdução à Programação 80h
  • Teoria Lógica de Programação 60h
  • Prática Banco de Dados I (SQL) 60h
2 SEM

2º Período — Estruturas

300h

Aprofundamento em POO, estruturas de dados e redes de computadores.

  • Prática Programação Orientada a Objetos 80h
  • Teoria Estrutura de Dados 60h
  • Prática Banco de Dados II (NoSQL) 60h
3 SEM

3º Período — Web & Software

300h

Desenvolvimento web, engenharia de software e primeiras arquiteturas.

  • Prática Desenvolvimento Frontend 80h
  • Prática Desenvolvimento Backend 80h
  • Teoria Arquitetura de Sistemas 60h
4 SEM

4º Período — Full-Stack

300h

Construção de aplicações completas com front-end, back-end e mobile.

  • Prática Desenvolvimento Mobile 80h
  • Teoria Segurança da Informação 60h
  • Prática Qualidade e Testes 60h
5 SEM

5º Período — Cloud & Dados

300h

Computação em nuvem, ciência de dados e arquiteturas escaláveis.

  • Prática Cloud Computing 80h
  • Teoria Inteligência Artificial 60h
  • Projeto Projeto Integrador I 60h
6 SEM

6º Período — Mercado & TCC

300h

Gestão de projetos, empreendedorismo e trabalho de conclusão.

  • Teoria Governança de TI 60h
  • Prática Big Data / Análise de Dados 60h
  • TCC Estágio Supervisionado + TCC 80h
